# Changelog

All notable changes to this project will be dcumented in this file (from now on).

## [0.3.0] - 2026-06-24
### Added
- Implemented an Incremental Search function in the message bar.

### Fixed
- Resolved a minor bug causing the message bar to not be updated when the user input is clear.
- Resolved a minor bug allowing multiple key strokes to be read from one key press.

## [0.2.2] - 2026-06-22
### Added
- Implemented line numbering in the main text editor view.

### Fixed
- Resolved a bug causing empty lines to not be cleared.

## [0.1.2] - 2026-06-12
### Added
- Initial release of the Mote text editor.
- C++ port of the original editor architecture.
- Custom piece table implementation to replace the standard `erow` line-based strcture.
